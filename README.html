ChromeXEditor - Chrome Extension Visual Editor

Overview

ChromeXEditor is a powerful web-based tool designed to streamline Chrome extension development through a visual interface.

Key Features

  • Visual file management system
  • Real-time code editing with syntax highlighting
  • Live preview functionality
  • Visual manifest.json editor
  • Dark/Light theme support
  • Auto-save capability
  • Project management system
  • One-click ZIP export

Getting Started

  1. Clone the repository
  2. Open index.html in a modern web browser
  3. Start creating or editing your Chrome extension

Requirements

  • Modern web browser (Chrome recommended)
  • Internet connection for CDN resources
  • Local storage enabled
