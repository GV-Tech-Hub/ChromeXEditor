ChromeXEditor - Chrome Extension Visual Editor
============================================

Description:
-----------
ChromeXEditor is a web-based visual editor designed specifically for Chrome extension development. It provides an intuitive interface for creating, editing, and managing Chrome extensions without the need for complex development environments.

Purpose:
--------
Created to simplify the Chrome extension development process, especially for developers who prefer a visual approach or are new to extension development. It eliminates the need to manually manage multiple files and configurations.

Features:
---------
- Visual file management
- Real-time code editing with syntax highlighting
- Live preview functionality
- Manifest.json visual editor
- Theme customization (Dark/Light modes)
- Auto-save functionality
- Project management system
- ZIP export for easy distribution

Requirements:
------------
- Modern web browser (Chrome recommended)
- Internet connection (for CDN resources)
- Local storage enabled

Installation:
------------
1. Clone the repository
2. Open index.html in a web browser
   OR
   Host the files on a web server

No additional installation or setup required - it's a standalone web application.

Usage Example:
-------------
A developer wants to create a Chrome extension that modifies webpage styling:
1. Open ChromeXEditor
2. Create necessary files (manifest.json, content.js, popup.html)
3. Use the visual editor to modify code
4. Preview changes in real-time
5. Export the extension as ZIP
6. Load the extension in Chrome

The entire process is handled through a single, intuitive interface without switching between different tools or editors.
